# DopaList 使用指南

## 快速开始

### 1. 开发模式运行

如果您想在开发环境中运行应用（支持热重载）：

```bash
# 安装依赖（首次运行）
pnpm install

# 启动 Electron 开发模式
pnpm electron:dev
```

这将启动一个带有开发者工具的桌面应用窗口。

### 2. 构建桌面应用

如果您想构建可安装的桌面应用：

#### Windows 用户

```bash
# 构建 Windows 安装包
pnpm electron:build:win
```

构建完成后，在 `release` 目录找到 `DopaList Setup 1.0.0.exe`，双击安装即可。

#### macOS 用户

```bash
# 构建 macOS 应用
pnpm electron:build:mac
```

构建完成后，在 `release` 目录找到 `DopaList-1.0.0.dmg`，双击打开并拖到应用程序文件夹。

#### Linux 用户

```bash
# 构建 Linux 应用
pnpm electron:build:linux
```

构建完成后，在 `release` 目录找到 `DopaList-1.0.0.AppImage` 或 `.deb` 文件。

## 功能使用

### 任务管理

#### 添加任务

1. 在顶部输入框输入任务标题
2. 选择截止日期（可选）
3. 选择优先级（P1/P2/P3）
4. 选择类别（如果已创建类别）
5. 按回车或点击"+"按钮添加

#### 编辑任务

1. 点击任务右侧的编辑图标（铅笔）
2. 修改任务标题、优先级、截止日期或类别
3. 点击勾选图标保存，或点击 X 取消

#### 完成任务

点击任务左侧的复选框，任务将被标记为完成，并可能触发多巴胺奖励！

#### 删除任务

点击任务右侧的删除图标（垃圾桶），任务将被移到回收站。

### 类别管理

#### 创建类别

1. 点击顶部的"类别管理"按钮
2. 输入类别名称（如"工作"、"学习"、"生活"）
3. 选择一个颜色
4. 点击"添加类别"

#### 编辑类别

1. 在类别管理对话框中，点击类别右侧的编辑图标
2. 修改名称或颜色
3. 点击"保存修改"

#### 删除类别

1. 在类别管理对话框中，点击类别右侧的删除图标
2. 确认删除（该类别下的任务不会被删除，但会失去类别标记）

#### 按类别筛选

在任务列表上方，点击类别按钮即可只显示该类别的任务。

### 多巴胺奖励系统

完成任务时，系统会随机触发奖励：

- **60% 概率**：正常完成，简单的勾选动画
- **30% 概率**：小奖励，弹出提示（如"休息5分钟"、"喝杯水"）
- **10% 概率**：超级大奖，全屏五彩纸屑效果（如"看一集剧！"、"今晚不加班！"）

#### 自定义奖励

1. 点击顶部的"奖励编辑"按钮（礼物图标）
2. 添加、编辑或删除奖励文案
3. 调整奖励权重（权重越高，出现概率越大）

### 回收站

1. 点击顶部的"回收站"按钮
2. 查看已删除的任务
3. 可以恢复任务或永久删除

### 导出数据

1. 点击顶部的"导出"按钮
2. 选择导出格式（Excel）
3. 数据将下载到本地

## 快捷键

- `Ctrl/Cmd + Q`：退出应用
- `Ctrl/Cmd + R`：重新加载
- `F11`：全屏切换
- `Enter`：在输入框中快速添加任务
- `Escape`：取消编辑

## 数据安全

所有数据都存储在本地的 localStorage 中，不会上传到任何服务器。您的数据完全私密和安全。

## 常见问题

### Q: 数据会丢失吗？

A: 数据存储在浏览器的 localStorage 中，只要不清除浏览器数据或卸载应用，数据就不会丢失。建议定期使用导出功能备份数据。

### Q: 可以在多台设备间同步吗？

A: 当前版本不支持云同步，数据仅存储在本地。您可以使用导出/导入功能在设备间迁移数据。

### Q: 如何备份数据？

A: 使用顶部的"导出"功能，将数据导出为 Excel 文件保存。

### Q: 奖励机制可以关闭吗？

A: 当前版本不支持关闭奖励机制，但您可以在奖励编辑中删除所有奖励，这样就只会显示正常完成动画。

### Q: 支持子任务吗？

A: 支持！点击任务右侧的"+"图标即可添加子任务。
